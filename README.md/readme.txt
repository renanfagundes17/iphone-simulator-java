# 📱 Projeto iPhone — Simulação em Java com POO e UML

Este repositório apresenta um projeto desenvolvido durante meus estudos em Engenharia de Software. O objetivo foi simular, em Java, as principais funcionalidades de um *iPhone, aplicando conceitos de **Programação Orientada a Objetos (POO)* e *modelagem UML*.

---

## *✅ Tecnologias e Conceitos Utilizados*

- *Java* (sintaxe, classes, objetos, métodos)
- *POO*: Encapsulamento, Abstração, Modularização
- *Interfaces*: Separação clara de responsabilidades
- *UML*: Diagrama de Classes para planejamento da estrutura
- *Git e GitHub*: Versionamento de código e portfólio profissional

---

## *📚 Principais Aprendizados*

- Criação e implementação de *interfaces Java*
- Uso de uma *classe concreta que implementa múltiplas interfaces*
- Organização de código com *pasta src e divisão de responsabilidades*
- Construção de *diagrama UML* para visualização da arquitetura do sistema
- Aplicação de *boas práticas de código limpo e modular*

---

## *⚙️ Funcionalidades Simuladas*

### 🎵 Reprodutor Musical
- tocar() — Toca uma música
- pausar() — Pausa a reprodução
- selecionarMusica(String) — Escolhe a música a ser tocada

### 📞 Aparelho Telefônico
- ligar(String) — Faz uma ligação
- atender() — Atende uma chamada
- iniciarCorreioVoz() — Ativa o correio de voz

### 🌐 Navegador de Internet
- exibirPagina(String) — Acessa uma URL
- adicionarNovaAba() — Abre uma nova aba
- atualizarPagina() — Atualiza a página atual

## *📌 Como Executar o Projeto*

1. *Clone o repositório:*
   bash
   git clone https://github.com/seu-usuario/iphone-simulator-java.git
   
2. *Compile os arquivos:*
   bash
   javac src/*.java
   
3. *Execute o programa:*
   bash
   java -cp src Main
